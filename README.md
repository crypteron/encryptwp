# EncryptWP
EncryptWP adds military-grade encryption and tamper protection to WordPress user data. 

## Beta Testing
**WARNING:** This plugin is in public beta. There may very well be bugs and plugin incompatibilities, especially when decrypting user->display_name. 
**Do not use on a production site and take a backup of your database before using**

For plugin updates during the beta process, use the [GitHub Updater Plugin](https://github.com/afragen/github-updater)

## Installation and Setup Instructions
For detailed instructions on how to use the plugin, please visit [https://www.crypteron.com/encryptwp/beta/](https://www.crypteron.com/encryptwp/beta/).
